# Practice

Assignment of practice(Chinese)

涵盖Python程序设计、代码规范、软件测试、任务管理、文档撰写和英文资源使用等能力训练；

涉及数据分析、交互计算、社区编程等科学计算和软件开发模式。

1. 软件开发基础：Python开发环境建立  

2. 版本管理及团队编程1：建立Github账户、基本操作、加入社区编程组等，  

3. 版本管理及团队编程2：Git、Github、EGit等
